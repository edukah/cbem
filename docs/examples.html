cBEM Class Name Examples

🧱 Basic Block + Element + Modifier

.product-card__title — The title element of the product-card block.
.product-card__image--rounded — A modified version of the image element (rounded style).
.product-card__price--discounted — The price element with a discounted variant.

🏗️ Block with Context

.home-feed_product-card — A product-card block used within a home-feed block.
.dashboard_user-card — A user-card block shown inside a dashboard block.
.modal_login-form — A login-form used in a modal component.

🧩 Context + Element + Modifier

.home-feed_product-card__image-container — The image-container element of product-card inside home-feed.
.dashboard_user-card__avatar — The avatar element of user-card in dashboard context.
.modal_login-form__input--error — An input with error modifier inside login-form in a modal.

⚙️ Advanced Multi-Context

.home-container_showcase_product-card__header — The product-card's header element used inside showcase, which is inside home-container.
.a_b_c__element--active — Example of deep nesting: block a → b → c, then an element with active modifier.

🚫 Invalid Anti-Patterns

.block__element__sub — ❌ Double `__` breaks clarity (no nested elements).
.block--mod--extra — ❌ Double `--` is invalid modifier chaining.
__element — ❌ Missing block name.
.block__--modifier — ❌ Empty element before modifier.
